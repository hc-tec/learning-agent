# 道岔互动课 AI 素材生产手册

> 版本：v0.1  
> 场景：铁路道岔入门互动课  
> 目标用户：没有铁路基础的新员工  
> 生产方式：在 AI 生图网站、AI 视频网站上手工生成素材，不调用 API  
> 核心原则：AI 网站是画室，人是导演，提示词是任务单，互动项目是放映厅。

## 1. 一句话目标

让新员工通过 3 张静态图和 5 条 10 秒以内短视频，看懂一件事：

```text
转辙机推动尖轨移动。
尖轨贴靠到不同位置。
车轮路径随之改变。
列车就能从直向线路进入侧向线路。
```

费曼式说法：

```text
尖轨像会动的小门。
转辙机像推门的手。
基本轨像不动的门框。
门的位置变了，车轮走的路也变了。
```

## 2. 已定产品与素材决策

### 2.1 教学结构

第一版采用完整互动版：

```text
3 张核心静态图
5 条短视频
15 张视频关键帧
每条视频 3 张关键帧
```

三张核心静态图：

```text
S01 直向状态
S02 转换中状态
S03 侧向完成状态
```

五条短视频：

```text
V01 尖轨
V02 转辙机
V03 基本轨
V04 观察转换
V05 侧向通过
```

### 2.2 互动流程

```text
第 1 张：S01 直向状态
任务：认识 3 个关键零件
热点：尖轨、转辙机、基本轨
规则：3 个热点都看完后，进入 S02

第 2 张：S02 转换中状态
任务：看整体正在变化
热点：观察转换
规则：看完后进入 S03

第 3 张：S03 侧向完成状态
任务：确认最终结果
热点：侧向通过 / 车轮走向
规则：看完后完成本小节
```

### 2.3 画面风格

```text
半写实 3D 教学图
16:9 横屏，1920x1080 起步
固定 45 度俯视方向
核心静态图使用中景
视频允许轻微局部放大
工程教材标注风
黄色高亮 + 白色标签 + 深色描边
```

说明：

- 不是现场照片。
- 不是抽象工程图纸。
- 不是卡通短视频风。
- 画面像一个摆在桌上的干净铁路道岔教学模型。

### 2.4 参考图一致性规则

核心素材必须像同一个道岔：

```text
3 张核心静态图
15 张视频关键帧
5 条核心视频
```

扩展素材以后可以只保持同一种教学风格，不强制和主线道岔完全一致。

### 2.5 标注规则

AI 可以生成中文、高亮、箭头和路径线，但必须少而准。

允许：

```text
尖轨
基本轨
转辙机
贴靠
分离
锁闭
车轮路径
直向
侧向
转换中
稳定边界
侧向通过
```

不建议在图里生成长句，例如：

```text
转辙机会推动尖轨移动，使车轮沿着不同方向前进。
```

原因：

```text
图里的字越长，越容易糊、错、歪、挡住关键部件。
短标签像路牌，长句像作文。
AI 更适合画路牌，不适合在视频里写作文。
```

### 2.6 干净版与标注版

核心静态图和视频关键帧都保留两版：

```text
annotated = 标注版，有中文、箭头、高亮，默认上课使用
clean = 干净版，无文字、无箭头、无高亮，作为后续再生产底稿
```

生成顺序：

```text
先生成标注版。
再从标注版反向生成干净版。
```

原因：

```text
第一目标是拿来正式教学，所以先做上课要用的黑板。
干净版是以后改字、换箭头、继续派生视频时用的备用底板。
```

## 3. 手工生产工作流

### 3.1 生产顺序

```text
1. 生成 S01、S02、S03 三张核心静态图
2. 生成 V01-V05 的 15 张关键帧
3. 用每条视频的 3 张关键帧生成 5 条短视频
4. 选出成品，复制到 selected/
5. 把最终素材导入互动项目
```

关键原则：

```text
先把 3 根大柱子立住，再装门窗。
不要先做视频，再从视频里倒推主图。
```

### 3.2 视频节奏

每条视频按 10 秒设计：

```text
0 - 1.5 秒：起始状态，停住，让学生认清对象
1.5 - 8.5 秒：教学慢动作，只保留高亮、箭头、路径线
8.5 - 10 秒：结果状态，停住，显示 1 到 2 个结果标签
```

三张关键帧时间点：

```text
0 秒：起始状态 + 看哪里
5 秒：动作进行中 + 高亮/箭头/路径线，尽量不放中文
10 秒：结果状态 + 结果标签
```

默认视频策略：

```text
先用标注版关键帧生成视频。
如果文字、标注乱得影响理解，再退回干净版关键帧重做。
```

可接受：

```text
字有一点糊
标签轻微抖动
箭头有一点不稳定
```

不可接受：

```text
术语写错
标签挡住尖轨和车轮接触位置
轨道结构明显变形
车轮脱离轨面
动作因果看不懂
```

### 3.3 尝试轮数

每个静态图或关键帧最多尝试 5 轮。

如果 5 轮还不行：

```text
不要继续抽卡。
先改提示词、参考图或素材目标。
```

结构好但中文一般：

```text
只要术语没错、没有严重遮挡，可以先拿去做视频。
```

视频动作好但中文有点糊：

```text
只要不误导、不挡关键动作，可以作为第一版成品。
```

## 6. prompt-log.md 模板

每个素材文件夹都有自己的 `prompt-log.md`。

模板：

```markdown
# Prompt Log

## Attempt 001

- 目标素材：
- 网站/模型：
- 参考图：
- 上传素材：
- 提示词：
- 负面提示词：
- 关键参数：
- 结果文件：
- 评价：可用 / 接近 / 废稿
- 备注：
```

记录原则：

```text
保存每一次尝试。
明显废稿可以删原文件，但日志要保留一句原因。
接近成功的失败稿要保留，因为它能告诉下一轮怎么改。
```

## 7. 固定术语表

第一版只使用这些词，不临时换名：

| 术语 | 用途 | 费曼解释 |
| --- | --- | --- |
| 尖轨 | 活动导向部件 | 会动的小门 |
| 基本轨 | 固定边界 | 不动的门框 |
| 转辙机 | 驱动设备 | 推门的手 |
| 贴靠 | 尖轨靠住基本轨 | 门关到位 |
| 分离 | 尖轨离开原位置 | 门开始打开 |
| 锁闭 | 到位并固定 | 门关好并扣住 |
| 车轮路径 | 车轮实际走的路线 | 车轮被带着走的路 |
| 直向 | 直线方向 | 沿主线走 |
| 侧向 | 分支方向 | 转入另一条线 |
| 转换中 | 道岔正在变状态 | 门正在动 |
| 稳定边界 | 车轮可靠依靠的边 | 不会乱动的边 |
| 侧向通过 | 车轮走入侧线 | 真的改路了 |
